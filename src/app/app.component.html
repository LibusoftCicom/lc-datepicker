Methods

Define allowed/disabled dates:

To configure range of available dates in calendar you can use config.MinDate and config.MaxDate, all dates out of that range are disabled. Default available range: 01.01.1900 - 31.12.2099

const config = new DatePickerConfig(); config.MinDate = { years: {{ fromDateObject.years }}, months: {{ fromDateObject.months }}, date: {{ fromDateObject.date }} }; config.MaxDate = { years: {{ toDateObject.years }}, months: {{ toDateObject.months }}, date: {{ toDateObject.date }} };

or to configure dates which are disabled you can use config.setDisabledDates( Array<MomentInput> )

const config = new DatePickerConfig(); config.setDisabledDates({{ randomDisabledDates | json }});

if is pressed and that day is in previously defined disabled list, calendar will pick next available date


Define allowed/disabled time ranges:

To configure disabled time ranges you can use config.addDisabledTimeRange, all times between that ranges are disabled

const config = new DatePickerConfig(); config.addDisabledTimeRange('00:00','07:59'); config.addDisabledTimeRange('14:00', '16:59'); config.addDisabledTimeRange('21:00', '23:59');

if is pressed and current time is in disabled time range, calendar will pick next available time


Navigation

It is possible to navigate through grid of each component, so each developer can implement keyboard or any other navigation using DatePickerConfig methods. Beside that now it is possible to dynamically change panel layout with already built in method `CalendarType`

const config = new DatePickerConfig(); config.navigateUp(); config.navigateDown(); config.navigateRight(); config.navigateLeft(); config.nextPage(); config.previousPage(); config.confirm(); config.close();

Example used in demo:
this.renderer.listen('document', 'keydown', (event: KeyboardEvent) => {
 if(this.config.isFocused()){
  event.preventDefault();
  if(event.keyCode == 38) { this.config.navigateUp(); }
  if(event.keyCode == 40) { this.config.navigateDown(); }
  if(event.keyCode == 39) { this.config.navigateRight(); }
  if(event.keyCode == 37) { this.config.navigateLeft(); }
  if(event.keyCode == 13) { this.config.confirm(); }
  if(event.keyCode == 27) { this.config.close(); }
  if(event.keyCode == 34) { this.config.nextPage(); }
  if(event.keyCode == 33) { this.config.previousPage(); }
}


Description

Pure Angular date and time picker component.

  • LC DatePicker is an Angular component that generates a datepicker calendar on your input element
  • Compatible with Angular 2+ up to Angular v6.0.0
  • Only dependencies are RxJS, MomentJS and Font Awesome
  • Customizable date format and language
  • Can be configured as time, date-time, date, month or year picker

Getting Started

  1. Fork and clone this repo
  2. npm install
  3. Open a terminal and type "npm run start"
  4. Open "http://localhost:4200" to browser

Installation

To install this component to an external project, follow the procedure:

  1. npm install @libusoftcicom/lc-datepicker

  2. skip this if SystemJS is not used as module loader

    System.config({
    paths: {
    'npm:': 'node_modules/'
    },
    map: {
    '@libusoftcicom/lc-datepicker' : 'npm:@libusoftcicom/lc-datepicker/bundles/lc-datepicker.umd'
    } })
  3. import {LcDatePickerModule} from '@libusoftcicom/lc-datepicker';
    @NgModule({
    declarations: [
    ...
    ],
    imports: [
    ...
    ,LcDatePickerModule
    ],
    providers: [
    ...
    ],
    bootstrap: [...] }) export class AppModule {}
  4. import {DatePickerConfig, ECalendarType} from '@libusoftcicom/lc-datepicker';
    @Component({
    ...
    }) export class AppComponent {
    private dateValue: string = null;
    public config = new DatePickerConfig();
    public calendarOpened: boolean = false;

    constructor() {
     // configuration is optional
     this.config.CalendarType = ECalendarType.Date;
     this.config.Localization = 'en';
     ...
    }

    public get Date() {
     return this.dateValue;
    }

    public set Date(value: string) {
     this.dateValue = value;
    } }

  5. Use the following snippet inside your template:

    <lc-datepicker [(opened)]="calendarOpened" [config]="config" [(date)]="Date"></lc-datepicker>

DatePicker config parameters

  • CalendarType: ECalendarType
  • Localization: String
  • MaxYear: Number
  • MinYear: Number
  • MaxMonth: Number
  • MinMonth: Number
  • MaxDay: Number
  • MinDay: Number
  • ConfirmLabel: String
  • PrimaryColor: String
  • FontColor: String
  • Format: Moment.MomentInput
  • setDisabledDates( Array<Moment.MomentInput> )
  • addDisabledTimeRange( start<Moment.MomentInput>, stop<Moment.MomentInput> )
