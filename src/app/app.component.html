Description

Simple Angular2 date picker.


Getting Started

  1. Fork and clone this repo
  2. npm install
  3. Open a terminal and type "npm serve"
  4. Open "http://localhost:4200" to browser

Installation

To install this component to an external project, follow the procedure:

  1. npm install lc/lc-date-picker
  2. import {LCDatePickerComponent} from '../lib/lc-date-picker/lc-date-picker.component';
  3. Use the following snippet inside your template: <lc-date-picker [(opened)]="CalendarOpened" [config]="config" [(date)]="setDate"p></lc-date-picker>
